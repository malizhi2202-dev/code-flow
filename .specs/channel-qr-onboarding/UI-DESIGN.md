## 组件设计

### QrScanModal（扫码弹窗）

```
┌─────────────────────────────────┐
│  📡 扫码接入飞书           ✕   │  ← 标题栏：渠道 icon + 名称 + 关闭按钮
│                                 │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  │     [ 二维码图片 ]       │    │  ← 200×200, 居中, 白色背景(bg-white)
│  │                         │    │     周围 8px 间距
│  └─────────────────────────┘    │
│                                 │
│  ⏱ 有效期剩余 4:32             │  ← 倒计时, font-mono, text-dim, 秒级更新
│  📱 请使用飞书 App 扫描二维码   │  ← 提示文字, font-size:13, text-secondary
│                                 │
│  ┌─────────────────────────────┐│
│  │ 🔄 重新获取                 ││  ← 次要按钮, ghost style
│  │ ✍️ 手动填写凭证             ││  ← 链接按钮, 切换到手动填表
│  └─────────────────────────────┘│
│                                 │
│  [状态变化时替换提示区]          │
│  ✅ 授权成功！渠道已激活         │  ← 成功: green, 1.5s 后自动关闭
│  ❌ 授权已取消                 │  ← 拒绝: danger, 显示「重试」按钮
│  ⏰ 二维码已过期               │  ← 过期: warning, 显示「重新获取」
│  ⚠️ 平台服务异常，请稍后重试    │  ← 错误: danger, 显示错误码 + 「重试」
└─────────────────────────────────┘
```

### 调性对齐

| 维度 | 取值 | 来源 |
|---|---|---|
| 背景 | `var(--bg-card)` | 工业暗色面板 |
| 边框 | `1px solid var(--border)` | 微边框风格 |
| 字体 | `var(--font-mono)` 用于倒计时；`var(--font-display)` 用于标题 | 等宽 + 工业 |
| 颜色 | 成功 `var(--green)` / 错误 `var(--red)` / 警告 `var(--color-warning)` | 既有语义色 |
| 二维码区 | `background: #fff`（白色底确保可扫） | 无障碍要求 |
| 圆角 | `borderRadius: 8` | 与项目 cards 一致 |
| 阴影 | `boxShadow: 0 4px 24px rgba(0,0,0,0.5)` | 弹窗层级 |

### 状态与 UI 映射

| OAuth 状态 | 倒计时 | 提示文字 | 操作按钮 | 二维码 |
|---|---|---|---|---|
| loading | — | 「正在获取二维码…」 | 无 | 占位 spinner |
| scanning | ⏱ 倒计时 | 「请使用 XX App 扫描二维码」 | 重新获取 / 手动填写 | ✅ 展示 |
| authorized | — | ✅「授权成功！渠道已激活」 | 无（1.5s 自动关闭） | 隐藏 |
| rejected | — | ❌「授权已取消」 | 🔄 重试 / ✍️ 手动填写 | 隐藏 |
| expired | — | ⏰「二维码已过期」 | 🔄 重新获取 / ✍️ 手动填写 | 隐藏 |
| error | — | ⚠️「平台服务异常（错误码: XXX）」 | 🔄 重试 | 隐藏 |
